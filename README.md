# SQL Data Cleaning Project

## Dataset
Layoffs Dataset

## Project Overview
This project focuses on cleaning and analyzing a layoffs dataset using SQL.

## Steps Performed

### 1. Data Cleaning
- Created a staging table
- Removed duplicate records
- Standardized company, industry, country, and date columns
- Handled null and blank values

### 2. Exploratory Data Analysis
- Total layoffs by company
- Total layoffs by industry
- Total layoffs by country
- Yearly layoffs analysis
- Monthly layoffs trends

## SQL Concepts Used
- SELECT
- WHERE
- GROUP BY
- HAVING
- ORDER BY
- CTEs
- JOINS
- Window Functions
- ROW_NUMBER()
- Aggregate Functions
